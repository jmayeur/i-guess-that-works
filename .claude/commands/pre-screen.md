Review and update the draft `$ARGUMENTS` looking for spelling, grammatical and clarity updates

## Steps

1. Read the draft file, if `$ARGUMENTS` are provided prompt to clarify what draft to edit

2. Find all spelling and grammatical errors, while it doesn't need to fully adhere to APA writing standards, ensure the good practices like period spacing, comma usage and other key formats are consistent and correct.

3. Prompt the user to see which fixes the would like to apply.

4. Apply only the approved changes

5. Produce a short summary of the intent of the draft and suggest any improvements to clarity, messaging or impact

6. Only apply updates if the user approves the suggestions


## Rules

- This draft content is intended to support both a LinkedIn Post and a Hugo blog post, do not suggest any markdown related changes as the content needs to work in LinkedIn. There is a separate command to convert the draft into a Hugo ready post.
- Do not change the voice of the draft, the goal is to highlight the message and remove mechanical errors.
- Do not suggest large content changes - the final output should not be flagged as AI generated.
- Do not make any changes to the draft unless the are explicitly approved.
